What this is

Cyber Threat Radar is a public-signal visualizer. Live mode tries public SANS ISC/DShield and CISA KEV feeds. If those feeds are blocked by CORS, rate limits or browser policy, the tool uses demo events so the map still works.

Why this is honest: classic cyber attack maps usually depend on private honeypots and vendor telemetry. This version does not claim exact origin, attacker identity or full internet coverage.

Future premium layer: saved watchlists, organization-specific ports, alert thresholds, webhook/email summaries and a backend feed cache.
